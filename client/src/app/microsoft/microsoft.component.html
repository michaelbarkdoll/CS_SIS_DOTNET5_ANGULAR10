Microsoft Dev Tools for Teaching

The School of Computing Computer Science Program is a member of the Microsoft Dev Tools for Teaching (formerly Microsoft Imagine, DreamSpark and Microsoft Developer Network Academic Alliance), which entitles us to install the Microsoft software products included in the MSDN Academic Alliance Subscription on our lab machines for instructional and non-commercial research use. CS faculty and CS students may install the software on their personal computers as well for teaching, study and research purpose at no cost. Each student is only allowed to obtain one copy and one key for each software.

Microsoft Azure Dev Tools for Teaching or simply Azure Dev Tools for Teaching is a Microsoft program to provide students with Microsoft software design, Microsoft developer tools, Cloud Computing Access and learning resources. The program is available for university/college and K-12 students Azure for Student and Azure Dev Tools for teaching are available in more than 140 countries.
